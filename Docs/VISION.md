# PROJECT VISION

This project is a high-quality naval vertical slice built in Unreal Engine 5.

The goal is to create a visually impressive, technically sound environment where:
- The player starts in a port
- Can enter a ship
- Can sail in a believable ocean environment

PRIORITY ORDER:
1. Visual quality (lighting, composition, realism)
2. System coherence (everything works together cleanly)
3. Stability (no broken interactions)
4. Expandability (future systems possible)

NOT A PRIORITY:
- Full game systems
- Multiplayer
- Large open world
- Complex economy

PHILOSOPHY:
Maximum perceived quality with controlled complexity.
