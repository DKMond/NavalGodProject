# TASK ID
MISSION_001_WORLD_BASE

## Assigned Agent
World Foundation Agent

## Mission Title
Create the base world substrate for the naval vertical slice

## Objective
Define and create the foundational world layer that future map content will sit on.

## Context
This is the first technical world-building task. It establishes the base environment required for future port, coast, ship, and navigation systems.

## Inputs
- Docs/VISION.md
- Docs/PIPELINE_RULES.md
- Docs/COMMAND_PROTOCOL.md
- Prompts/WORLD_AGENT.md

## Deliverables
- A recommended world foundation approach
- Base ocean or terrain substrate plan
- Proposed world scale approach
- Proposed boundary logic
- Initial collision and navigability assumptions

## Definition of Done
- A clear world foundation proposal exists
- Scale assumptions are explicit
- Boundaries are defined conceptually
- Risks and next dependencies are identified

## Constraints
- Must support a naval vertical slice
- Must remain visually believable
- Must be performant on current hardware
- Must not assume a giant open world

## Forbidden Actions
- Do not invent gameplay systems
- Do not define lighting as final
- Do not expand into UI or narrative systems

## Dependencies
This task depends on the project vision and pipeline rules.
Future port, ship, and gameplay systems depend on this task.
